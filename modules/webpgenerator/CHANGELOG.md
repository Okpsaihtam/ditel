# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).
## [1.0.4] - 2019-04-02

### Added
- Set rewrite base for webp images
- Fix binaries permissions during install
- Add 'missing fileinfo extension' message to config controller
- Also convert the original image

## [1.0.3] - 2019-02-14
- Only show webp images when the module is enabled
- Add allow from all to generated htaccess
- Implement demo mode functionality (useful when the images are generated)
- Improve old PrestaShop functionality

## [1.0.2] - 2019-01-12
### Bugfix
- Add AdminWebpgeneratorConfig controller to the installable tabs

## [1.0.1] - 2019-12-11
### Added
- Improve compatibility with Prestashop 1.6 versions

## [1.0.0] - 2019-12-11
